## The promise
Atlas turns raw ideas into living, working projects. Not an AI coding app. Not a PM tool. Not a chatbot. "You bring the vision. Atlas helps make it real."

## Phase 1 — the ONLY three things that matter
1. Turn an idea into a project (Understand it)
2. Turn the project into something working (Build it)  
3. Continue building from where they left off (Evolve it)

Thought → Reality → Continuity.

## The Founder Filter (apply before every feature)
1. Does this help a person understand their idea?
2. Does this help a person build something working?
3. Does this help a person continue later without starting over?
No to all three = waits.

## What Atlas will NOT do in Phase 1
- No multiple environments. One environment. One source of truth. No split architecture.
- No infrastructure before proving the core loop.
- No feature to compensate for another broken feature.
- No third-party complexity users must understand (GitHub, Cloud Run, Supabase, Vercel, OAuth).
- No ten simultaneous systems. One active mission.
- Working > Elegant. Always.
- No preserving old decisions just because time was invested.

## Asset survival table (key items)
- Visual identity (Luxury Obsidian): ✅ Keep idea + code
- Project continuity: ✅ Keep idea, ⚠️ re-earn code
- Memory philosophy: ✅ Keep idea, ❌ rebuild implementation
- Workspace UI: ✅ Keep idea, ⚠️ evaluate against Phase 1
- Blueprints/Artifacts/Diff viewer: ⏸️ Phase 2
- Auth systems: ❌ start clean
- Four-repo architecture: ❌ root cause of chaos
- Ghost databases: ❌ archive

## Two builders — never confuse them
- Builder 1: You are building Atlas
- Builder 2: Atlas builds projects for users
These are separate systems. Confusion between them caused most of the drift.

**Why:** The constitution was written after diagnosing that the root cause of all failures was over-connection across too many systems, not broken code.

**How to apply:** Before writing any code, check against Phase 1 actions. ATLAS_ZERO.md at workspace root wins over any code, feature, or conversation that conflicts with it.
